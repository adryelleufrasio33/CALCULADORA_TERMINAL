
# Calculadora Simples no Terminal

Estava vasculhando meu computador e encontrei este código que desenvolvi no primeiro semestre da faculdade de Análise e Desenvolvimento de Sistemas.

Resolvi subir este projeto porque ele marca uma fase inicial importante da minha trajetória com programação. Mesmo sendo um código de iniciante, já dá para perceber uma preocupação com fluxo lógico, estruturas de repetição, condicionais e organização básica das operações.

Como acontece com muita gente no começo da jornada, este foi um dos meus exercícios de calculadora no terminal, abordando operações simples como:

- soma
- subtração
- multiplicação
- divisão

Mais do que um projeto “complexo”, este repositório representa um registro de evolução. Ele mostra como eu já estava começando a construir raciocínio lógico e controle de fluxo desde os primeiros contatos com Python.

## Objetivo

Registrar uma etapa do meu aprendizado e manter esse projeto como parte da minha evolução na programação.

## Tecnologias utilizadas

- Python

## Observação

Este projeto foi feito no início da graduação, então ele reflete meu nível de conhecimento naquele momento. Ainda assim, foi um exercício importante para fortalecer a base de lógica de programação.
